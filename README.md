# Fancim Crawler

## 프로젝트 소개

일리오라는 기업과 협업한 프로젝트입니다. 유튜브, 트위치, 아프리카를 크롤링하여 데이터를 수집해 일리오에 등록된 BJ들의 흥망성쇠를 나타내는 프로젝트입니다.

## Start

```
npm install
```

## Stacks

- Javascript
- React.js
- Antdesign
- Python
- Django
- Mysql
- Selenium
